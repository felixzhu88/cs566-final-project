### Run Locally

If you want to run the website on your local machine:

**1.****Prerequisites:******
**-** Node.js (v14 or higher)
**-** npm (comes with Node.js)

**2.****Clone and navigate to frontend:******

```bash
   git clone https://github.com/felixzhu88/cs566-final-project.git
   cd cs566-final-project/frontend/drowsiness-detection
```

**3.****Install dependencies:******

```bash
   npm install
```

**4.****Run the development server:******

```bash
   npm start
```

**5.****View in browser:******
   Open **[**http://localhost:3000**](**http://localhost:3000**)**
